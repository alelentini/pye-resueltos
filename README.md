# Problemas y Ejercicios Resueltos de Probabilidad y Estadística

:warning: **Trabajo en progreso**

El presente trabajo contiene la resolución de ejercicios y problemas correspondientes a las guías de TP de la asignatura Probabilidad y Estadística de la Universidad Tecnológica Nacional - Regional Haedo, así como también incluye resoluciones a ejercicios y problemas de otras fuentes (exámenes parciales y finales, libros de texto, etc.).
